# vfa-golang-gin-framework

GIN framework

## Requirement

* Go 1.93
* Postgresql
* golang-jwt
* godotenv
* gorm

## RUN
go run main.go

## API
- POST: http://localhost:8000/register
- POST: http://localhost:8000/login
- POST: http://localhost:8000/api/entry
- GET: http://localhost:8000/api/entries
- GET: http://localhost:8000/api/entry/:entry_id
